# Homebrew Tap for Stik

Instant thought capture for macOS.

## Install

```bash
brew install --cask 0xMassi/stik/stik
```

Or add the tap first:

```bash
brew tap 0xMassi/stik
brew install --cask stik
```

## About

[Stik](https://github.com/0xMassi/stik_app) is a lightweight note-capture app for macOS. Press a shortcut, type your thought, close. Under 3 seconds.
